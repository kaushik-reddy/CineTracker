## üöÄ How to Use

### Getting Started
1.  **Sign Up**: Create an account to start building your library.
2.  **Add Media**: Search for titles or manually add movies, series, and books to your library.
3.  **Schedule**: Pick a date and time for your next watch.
4.  **Watch**: 
    - Click "Watch Now" to track progress manually.
    - Or "Create Watch Party" to invite friends.

### Key Features
- **Library Management**: Filter by genre, platform, status (Not Started, In Progress, Completed).
- **Timeline View**: A calendar view of your past and upcoming entertainment schedule.
- **Walkthrough Mode**: New users get a guided tour of the features.
- **PWA Support**: Install as an app on your mobile device for native-like experience.
